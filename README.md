# lints
## Focus CB ued eslint config base on eslint:recommend
## Usage
 - npm install eslint eslint-config-cbued --save-dev
 - 在项目根目录下新建或者编辑 .eslintrc.js 
 - 将以下内容填入
 ```javascript
 module.exports = {
  "extends": "cbued"
 };
 ```
